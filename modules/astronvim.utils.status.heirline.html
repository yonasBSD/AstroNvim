A helper function to get the type a tab or buffer is

Parameters:

  • self table the self table from a heirline component function
  • prefix ? string the prefix of the type, either "tab" or "buffer" (Default: "buffer")

Returns:

    string # the string of prefix with the type (i.e. "active" or "visible")

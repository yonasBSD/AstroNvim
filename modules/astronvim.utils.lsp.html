Module astronvim.utils.lsp

AstroNvim LSP Utils

LSP related utility functions to use within AstroNvim and user configurations.

This module can be loaded with local lsp_utils = require("astronvim.utils.lsp")

has_capability(capability, filter)
Helper function to check if any active LSP clients given a filter provide a specific capability

Parameters:

  • capability string The server capability to check for (example: "documentFormattingProvider")
  • filter

    vim.lsp.getactiveclients.filter|nil (table|nil) A table with

              key-value pairs used to filter the returned clients.
          The available keys are:
           - id (number): Only return clients with the given id
           - bufnr (number): Only return clients attached to this buffer
           - name (string): Only return clients with the given name

Returns:

    boolean # Whether or not any of the clients provide the capability

  • bufnr number The buffer that the LSP client is attaching to

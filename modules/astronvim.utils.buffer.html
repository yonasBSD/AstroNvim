Module astronvim.utils.buffer

AstroNvim Buffer Utilities

Buffer management related utility functions

This module can be loaded with local buffer_utils = require "astronvim.utils.buffer"

is_valid(bufnr)
Check if a buffer is valid

Parameters:

  • bufnr number The buffer to check

Returns:

    boolean # Whether the buffer is valid or not
move(n)
Move the current buffer tab n places in the bufferline

Parameters:

  • n number The number of tabs to move the current buffer over by (positive = right, negative = left)
nav(n)
Navigate left and right by n places in the bufferline

Parameters:

  • n number The number of tabs to navigate to (positive = right, negative = left)
nav_to(tabnr)
Navigate to a specific buffer by its position in the bufferline

Parameters:

  • tabnr number The position of the buffer to navigate to
close(bufnr, force)
Close a given buffer

Parameters:

  • bufnr ? number The buffer to close or the current buffer if not provided
  • force ? boolean Whether or not to foce close the buffers or confirm changes (default: false)
close_all(keep_current, force)
Close all buffers

Parameters:

  • keep_current ? boolean Whether or not to keep the current buffer (default: false)
  • force ? boolean Whether or not to foce close the buffers or confirm changes (default: false)

Close buffers to the left of the current buffer

close_right(force)
Close buffers to the right of the current buffer

sort(compare_func, skip_autocmd)
Sort a the buffers in the current tab based on some comparator

Parameters:

  • compare_func string|function a string of a comparator defined in require("astronvim.utils.buffer").comparator or a custom comparator function
  • skip_autocmd boolean|nil whether or not to skip triggering AstroBufsUpdated autocmd event

Returns:

    boolean # Whether or not the buffers were sorted

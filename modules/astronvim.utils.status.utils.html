Module astronvim.utils.status.utils

AstroNvim Status Utilities

Functions

build_provider(opts, provider)
Convert a component parameter table to a table that can be used with the component builder

Parameters:

  • opts ? table a table of provider options
  • provider ? function|string a provider in M.providers

Returns:

    table|false # the provider table that can be used in M.component.builder
setup_providers(opts, providers, setup)
Convert key/value table of options to an array of providers for the component builder

Parameters:

  • opts table the table of options for the components
  • providers string[] an ordered list like array of providers that are configured in the options table
  • setup ? function a function that takes provider options table, provider name, provider index and returns the setup provider table, optional, default is M.build_provider

Returns:

    table # the fully setup options table with the appropriately ordered providers
width(is_winbar)
A utility function to get the width of the bar

Parameters:

  • is_winbar ? boolean true if you want the width of the winbar, false if you want the statusline width

Returns:

    integer # the width of the specified bar
pad_string(str, padding)
Add left and/or right padding to a string

Parameters:

  • str string the string to add padding to
  • padding table a table of the format { left = 0, right = 0} that defines the number of spaces to include to the left and the right of the string

Returns:

    string # the padded string
stylize(str, opts)
A utility function to stylize a string with an icon from lspkind, separators, and left/right padding

Parameters:

  • str ? string the string to stylize
  • opts ? table options of { padding = { left = 0, right = 0 }, separator = { left = "|", right = "|" }, escape = true, show_empty = false, icon = { kind = "NONE", padding = { left = 0, right = 0 } } }

Returns:

    string # the stylized string

Usage:

    local string = require("astronvim.utils.status").utils.stylize("Hello", { padding = { left = 1, right = 1 }, icon = { kind = "String" } })
surround(separator, color, component, condition)
Surround component with separator and color adjustment

Parameters:

  • separator string|string[] the separator index to use in env.separators
  • color function|string|table the color to use as the separator foreground/component background
  • component table the component to surround
  • condition boolean|function the condition for displaying the surrounded component

Returns:

    table # the new surrounded component

Get the null-ls sources for a given null-ls method

statuscolumn_clickargs(self, minwid, clicks, button, mods)
A helper function for decoding statuscolumn click events with mouse click pressed, modifier keys, as well as which signcolumn sign was clicked if any

Parameters:

  • self any the self parameter from Heirline component on_click.callback function call
  • minwid any the minwid parameter from Heirline component on_click.callback function call
  • clicks any the clicks parameter from Heirline component on_click.callback function call
  • button any the button parameter from Heirline component on_click.callback function call
  • mods any the button parameter from Heirline component on_click.callback function call

Returns:

    table # the argument table with the decoded mouse information and signcolumn signs information

Usage:

    local heirline_component = { on_click = { callback = function(...) local args = require("astronvim.utils.status").utils.statuscolumn_clickargs(...) end } }

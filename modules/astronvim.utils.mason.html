Module astronvim.utils.mason

Mason Utils

Mason related utility functions to use within AstroNvim and user configurations.

This module can be loaded with local mason_utils = require("astronvim.utils.mason")

Functions

update(pkg_names, auto_install)
Update specified mason packages, or just update the registries if no packages are listed

Parameters:

  • pkg_names ? string|string[] The package names as defined in Mason (Not mason-lspconfig or mason-null-ls) if the value is nil then it will just update the registries
  • auto_install ? boolean whether or not to install a package that is not currently installed (default: True)

Module astronvim.utils.status.hl

AstroNvim Status Highlighting

Statusline related highlighting utilities

This module can be loaded with local hl = require "astronvim.utils.status.hl"

lualine_mode(mode, fallback)
Get the highlight background color of the lualine theme for the current colorscheme

Parameters:

  • mode string the neovim mode to get the color of
  • fallback string the color to fallback on if a lualine theme is not present

Returns:

    string # The background color of the lualine theme or the fallback parameter if one doesn't exist

get_attributes(name, include_bg)
Merge the color and attributes from user settings for a given name

Parameters:

  • name string, the name of the element to get the attributes and colors for
  • include_bg ? boolean whether or not to include background color (Default: false)

Returns:

    table # a table of highlight information

Usage:

    local heirline_component = { provider = "Example Provider", hl = require("astronvim.utils.status").hl.get_attributes("treesitter") },
